## Requirements
+ Apache Webserver
+ PHP
+ MySQL Database

## Installation
+ Import *sail-final.sql* into your MySQL Server
+ Delete *INSTALL/* Directory
+ Setup credentials in *function.php* on line 2
+ Move files to Webserver
+ Ready to plan
